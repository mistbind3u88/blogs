# なにが入っている？

Issue / PR をもとに作った活動サマリをまとめています。

## どう整理している？

- 年次サマリは年ごとに分けています。
- Issue は closedAt の年で集計しています。
- PR は mergedAt の年で集計しています。
- カテゴリは Issue / PR のタイトルに含まれるラベル風の文字列をもとに整理しています。
- 半期サマリはファイル名だけで期間が分かるように、half-year 直下に置いています。

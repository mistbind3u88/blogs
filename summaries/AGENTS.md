# 活動サマリ記事向けガイド

## 記事の扱い

- 年次サマリと半期サマリでは、対象期間、集計基準、取得時点を明示する。
- 取得できていない情報がある場合は、未取得であることと影響範囲を本文に書く。
- 件数だけでなく、活動の流れやテーマの切れ目が追える構成を優先する。

## 更新時の確認

- `summaries/README.md` のリンク一覧と、追加・更新したサマリファイルの対応を確認する。
- 件数、差分量、代表トピック、Open Issue / PR は、同じ基準で比較できるようにする。
- 自動生成された要約でも、読み手が活動の流れを追えるようにテーマ単位で整理する。

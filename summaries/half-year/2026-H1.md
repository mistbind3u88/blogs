# 2026-H1 半期サマリ

## なにやってたの？

2026-H1 は、三題噺まわりの進め方を見直しながら、その運用を支える仕組みも整えていた時期でした。
レビュー四点セットでは、レビュー、批評、逆批評、総論をどう分けて書くかを考え直し、複数のレビュー記事をまとめて更新していました。あわせて、三題噺向けのスキルや AGENTS、Issue と Pull Request の書き方も見直していて、作品そのものと、その周辺の流れを一緒に整えていました。

それだけでなく、Node.js まわりの依存更新や整形・lint の安定化も進めていました。
この半期は、新しく書くことだけに集中するというより、「あとから同じように回せる形にする」ところまで含めて手を入れていた時期だったと言えます。

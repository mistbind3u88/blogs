#### 中心ノード（3語の共通軸）

中心ノードは「山崩れ・料理人・アルバム」を同じ文法で並べるための変換軸。
基本の読み方は、各行を「山崩れ = 左 / 料理人 = 中央 / アルバム = 右」と対応させる。
厳密な一対一対応ではなく、物語化のために役割を入れ替えたり反転して読んでもよい。
目的は正解探しではなく、3語を同一フレームで接続して飛躍の足場を作ること。

- 崩れる / 作る / 残す
- 自然 / 技術 / 記録
- 破壊 / 維持 / 継承
- 偶然 / 手業 / 編集
- 現場 / 厨房 / アーカイブ
- 土 / 火 / 紙（またはデータ）
- 一回性 / 再現性 / 反復性
- 生存 / 生活 / 物語化
- 身体感覚 / 味覚設計 / 記憶想起
- 喪失 / 応答 / 収蔵
- 沈むもの / 温めるもの / 留めるもの
- 無秩序 / 段取り / 配列
- 事故 / 仕事 / 作品
- 危機管理 / 品質管理 / 情報管理
- 時間の断絶 / 時間の運用 / 時間の保存
- 地形の変化 / 関係の調律 / 意味の再編集
- 目撃 / 提供 / 証言
- 忘却 / もてなし / 記録
- 取りこぼし / 掬い上げ / 綴じ込み
- その場で消える価値（料理）と後に残る価値（アルバム）をどう接続するか

#### 山崩れ

##### 意味の芯

- 地形が崩落する自然現象
  - 比喩ジャンプ: 「地図が突然、嘘になる瞬間」
  - 異分野ジャンプ: サーバ障害で冗長化が破れ、全系が落ちる状態
  - 視点反転: 山が崩れるのでなく、人間側の「前提」が崩れる
- 突発性
  - 比喩ジャンプ: 会話の一語で関係が終わる瞬間
  - 異分野ジャンプ: 相場のフラッシュクラッシュ
  - 視点反転: 前触れは常にあったが、受信側が未接続だった
- 不可逆性
  - 比喩ジャンプ: 既読がついたメッセージは未読に戻せない
  - 異分野ジャンプ: 焼成後の陶器は土に戻らない
  - 視点反転: 戻れないこと自体を新しい設計条件にする
- 境界の喪失（山道・集落・生活圏）
  - 比喩ジャンプ: 公私の境目が溶けるリモート時代
  - 異分野ジャンプ: ジャンル混交で棚分け不能な作品
  - 視点反転: 失うのは境界でなく、境界に依存した思考
- 上から下への流動
  - 比喩ジャンプ: 組織上層の一言が現場を押し流す
  - 異分野ジャンプ: バズが下層コミュニティへ雪崩れ込む
  - 視点反転: 流れているのは土砂でなく意思決定の責任
- 埋没と露出（隠れていたものが出る）
  - 比喩ジャンプ: 事故後にだけ露見する本音
  - 異分野ジャンプ: 考古学的発掘としての災害現場
  - 視点反転: 失われたのでなく、見え方が変わっただけ

##### 連想拡張

- 雨・地震・地鳴り・土の匂い
  - 感覚ジャンプ: 匂いを「時報」として扱う
  - メディアジャンプ: 音だけで場面を成立させる脚本
  - 記憶ジャンプ: 匂いが現在より過去を先に開く
- 警報・避難・孤立
  - 比喩ジャンプ: 既読はつくのに会えない関係
  - 異分野ジャンプ: オンライン接続と心理的孤立の逆転
  - 視点反転: 孤立は地理でなく、意思決定の分断
- 復旧・土木・地質
  - 比喩ジャンプ: 壊れた関係の「工事計画書」
  - 異分野ジャンプ: カウンセリングを地盤改良として描く
  - 視点反転: 復旧は元に戻すのでなく、前より強く作る
- 「崩れる」の比喩（信頼、計画、感情）
  - 比喩ジャンプ: 微笑みのまま内側だけ崩壊する人物
  - 異分野ジャンプ: ヒットチャート上位でもバンド解散前夜
  - 視点反転: 崩壊は敗北でなく再編集の開始合図
- 「地盤」の比喩（価値観、共同体、家族）
  - 比喩ジャンプ: 家訓が消え、献立が家の憲法になる
  - 異分野ジャンプ: コミュニティ規約の改訂を地盤補強として描く
  - 視点反転: 足場は「場所」でなく「反復する行為」

#### 料理人

##### 意味の芯

- 料理を職能として作る人
  - 比喩ジャンプ: 火と時間で翻訳する通訳者
  - 異分野ジャンプ: データ整形エンジニアとしての料理人
  - 視点反転: 作る人でなく、場の温度を調律する人
- 技術と再現性
  - 比喩ジャンプ: 同じ曲を毎夜違う客席に届ける演奏家
  - 異分野ジャンプ: CI/CDで安定デプロイする感覚
  - 視点反転: 再現はコピーでなく、条件差の吸収
- 火・刃物・時間管理
  - 比喩ジャンプ: 包丁は編集、火は締切
  - 異分野ジャンプ: 手術室のような緊張管理
  - 視点反転: 危険物を扱うのでなく、危険を「順番」で無害化する
- 味覚の設計
  - 比喩ジャンプ: 料理を「読後感」で設計する
  - 異分野ジャンプ: プレイリストの曲順設計に等置する
  - 視点反転: 味は舌でなく会話の速度で評価される
- 衛生・段取り・責任
  - 比喩ジャンプ: 清潔は倫理の可視化
  - 異分野ジャンプ: セキュリティパッチ運用の思想
  - 視点反転: 責任は失敗時でなく平時の反復に宿る
- 客に届く最終工程の担い手
  - 比喩ジャンプ: 皿を置く行為は句点を打つ行為
  - 異分野ジャンプ: ライブ配信の遅延調整に近い
  - 視点反転: 提供は終点でなく、物語開始の合図

##### 連想拡張

- 見習い・親方・店主

  - 異分野ジャンプ: ローカルキャッシュ戦略としての保存食

#### アルバム

##### 意味の芯

  - 異分野ジャンプ: データベースのビュー設計
  - 視点反転: 記録の器でなく、記憶を誤読させる装置
- 写真アルバム（私的記憶）
  - 比喩ジャンプ: 不在者のポートレート集
  - 異分野ジャンプ: 防犯カメラ映像との主観差
  - 視点反転: 写真は真実保存でなく、感情の演出記録
- 音楽アルバム（作品単位）
  - 比喩ジャンプ: 一冊の詩集として聴く
  - 異分野ジャンプ: コース料理の提供順
  - 視点反転: 曲は主役でなく、無音区間が意味を作る
- 断片の配列による意味化
  - 比喩ジャンプ: 切り抜き新聞で別事件が生まれる
  - 異分野ジャンプ: モンタージュ理論
  - 視点反転: 断片が物語を作るのでなく、欠落が物語を駆動
- 時系列の操作
  - 比喩ジャンプ: 昨日を明日の伏線にする
  - 異分野ジャンプ: ノンリニア編集タイムライン
  - 視点反転: 過去回想でなく、未来からの逆照射
- 保存と選別

  - 異分野ジャンプ: アーカイブ学と廃棄ポリシー
  - 視点反転: 保存は喪失の対義語でなく、喪失の設計

#### 横断リンク（語と語の橋）

横断リンクは、中心ノードで並べた3語のあいだに起こる「接点・摩擦・交換」を具体化する層。
ここでは抽象語よりも、発想を広げやすい関係（連結・偏位・余白）を優先して書く。
目的は、三題を単に並置せず、因果や緊張として接続すること。

- 山崩れ x 料理人: 供給断絶、限られた食材、非常時の調理、共同体維持
  - 連結: 「崩れた地形」が「献立の制約」に直結する
  - 偏位: 料理人を供給者でなく「地域の温度管理者」として捉える
  - 余白: 何をどこまで食として提供するかの線引きは未定のまま残す
- 山崩れ x アルバム: 災害記録、失われる記憶、泥から出る写真、証言資料
  - 連結: 「埋もれたものが現れる」現象と「記録が現れる」行為を重ねる
  - 偏位: アルバムを追悼の道具でなく、地層のような情報層として読む
  - 余白: どの記録を私的に保ち、どれを公に開くかは決めきらない
- 料理人 x アルバム: レシピ帳、料理写真集、味の記憶、コース構成=曲順
  - 連結: 料理の手順とアルバムの配列を同じ「編集操作」として扱う
  - 偏位: 皿を味覚体験でなく「ページ」として読む
  - 余白: 再現重視か即興重視かは場面ごとに揺らがせる
- 3語同時: 「崩壊の現場」で「生をつなぐ技」が「記憶として編まれる」
  - 連結: 崩壊（山）・調理（人）・保存（媒体）を一連の処理として並べる
  - 偏位: 三題を「災害の話」ではなく「編集の話」として読む
  - 余白: 何を最終的な主題に据えるかは最後まで固定しない

#### 抽象化ノード（さらに広げる軸）

抽象化ノードは、横断リンクで生じた出来事を上位概念（時間・身体・倫理など）で再整理する層。
個別の場面をテーマへ引き上げ、別の物語案にも転用できる形にする。
目的は、発想を「その場の思いつき」で終わらせず、構造として再利用可能にすること。

- 時間軸: 瞬間（崩落）/現在進行（調理）/保存（アルバム）
  - 連結: 同じ出来事を「発生時」「処理時」「保存時」で見分ける
  - 偏位: 直線時間でなく、反復閲覧される時間として捉える
  - 余白: どの時点を起点に語るかを確定しない
- 身体軸: 土に触れる手/包丁を握る手/ページをめくる手
  - 連結: 「触れる手」という共通動作で3語を束ねる
  - 偏位: 手を労働器官でなく、編集インターフェースとして見る
  - 余白: 誰の手を中心に置くかは複数候補のまま保持する
- 倫理軸: 自然災害の責任、職能の責任、記録者の責任
  - 連結: それぞれの責任を「行為の責任」として横並びにする
  - 偏位: 断罪の倫理でなく、運用の倫理として捉える
  - 余白: 責任の重みづけを一意に決めない
- 認識軸: 現実（災害）と表象（写真・音）
  - 連結: 体験と記録を「同一事象の別レイヤー」として扱う
  - 偏位: 記録を再現でなく、再構成として読む
  - 余白: どこまでを事実、どこからを解釈とするか固定しない
- 感情軸: 恐怖、安堵、追悼、継承
  - 連結: 感情の推移を「個人→共同体」の回路として捉える
  - 偏位: 感情を結果でなく、編集される素材として扱う
  - 余白: どの感情を主旋律にするかは保留する
